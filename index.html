01 / 概览

平地看不出本事，复杂里才见真章

本站合并承载两道机器人运动控制赛题。它们都叫「复合地形长距离穿越」，但地形不同、赛制与计分规则各自独立、互不混用—— 一个考带臂足的全身运控与 locomotion，一个考人形的长距离地形适应与抗冲击稳定。

02 / TronCamp · TRON 2

带臂足 · 复合地形长距离穿越

带臂足机器人（参考构型 TRON 2：双脚掌 / 轮 + 单机械臂）在约 100m、宽 3m 的复合地形直线赛道上比拼全身运动能力。赛道串联坡道、楼梯与 3 个不可推动高台；手臂可主动撑、扶、攀辅助通过并额外加分，鼓励「行走 + 操作」的统一策略。

本体：TRON 2 双脚掌 / 轮 + 单机械臂（6 DoF + 二指夹爪），自带下视 RGB-D + 前视 RGB-D + 激光雷达；具体传感器外参可参考选手资源页。

段名称长度关键参数
S1平地起步段20 m平整地面，无特殊要求
S2上坡段15 m坡度 +15°
S3下坡段15 m坡度 −15°
S4平地过渡段6 m平地缓冲
S5上楼梯段10 m阶高 10–20 cm（随机），阶宽 30 cm，共 6–10 级
S6下楼梯段10 m对称布置
S7平地段10 m平整地面
S8变台跨越段17 m3 个高台，高度依次 20 / 30 / 40 cm，台面 0.8m × 1.0m，不可推动（只能跨越）
终点终点段4 m平地，含终点感应线

计分（满分 100 · 距离 × 速度 + 协同奖励）

主成绩看走多远、多快；在此之上，locomotion 的协同行为（不跌倒 / 跌倒自恢复、手臂辅撑攀台）额外加分，越界扣分。

基础分 = D_eff / 100 m × 60 （满分 60；D_eff = 躯干质心沿赛道方向累计有效前程，不计回退，上限 100 m）
时间加成 = clamp(20 × (1 − (T − 240) / 240), 0, 20) （满分 20；T ≤ 240s 满分，≥ 480s 为 0；未完赛 T = 600s）
协同奖励 / 罚则（协同奖励上限 +20）检测方式分值
跨越高台时手臂辅撑攀爬末端施力 + 高度变化+4 / 台（≤ 3 台）
未跌倒 / 跌倒后自主爬起（二者取一）全程零跌倒 / 跌倒自恢复+5
越出 3m 计分走廊（|y| > 1.5m）每次越界−3 / 次

机型：TRON 2 腿式 sfyg（得分权重 ×1.0） / 轮式 wfyg（得分权重 ×0.8），含 REAL / ORIG 传感变体。单局限时 600s，跌倒不终止、可自主恢复。

03 / Humanoid · Oli

人形 · 复合地形长距离穿越

人形机器人 Oli EDU 在约 65m、宽 5m 的复合地形赛道上比拼长距离穿越。赛道串联坡道、坎、楼梯与沟壑，考察深度视觉感知、地形适应与抗冲击稳定。本赛题与 Tron 赛题的地形 / 赛制与计分规则完全独立，互不混用（最终得分计入构型权重后统一排名）。

本体：Oli EDU，自带头部 Realsense D435i；具体传感器外参可参考选手资源页。

段名称长度关键参数
S1平地边段9 m平整起步，无需转向
S2上坡段7.5 m坡度 +10°
S3斜坡直段5 m坡顶平台
S4下坡段7.5 m坡度 −10°
S5坎13 m3 个坎，高 0.2 / 0.25 / 0.3 m，宽 0.15 / 0.2 / 0.25 m，间距 3 m
S6上楼梯段4.5 m15 级，阶高 0.15 m、踏深 0.30 m
S7沟壑段4.75 m3 道沟，宽 0.2 / 0.25 / 0.3 m，沟间地板段宽 1.0 m
S8下楼梯段4.5 m15 级下降
终点终点段9.25 m平地，含终点感应线

计分（满分 100 · 距离 × 速度）

主成绩看在 65m 复合地形上走多远、多快；触地打滑、摔倒、越界扣分。

基础分 = D_eff / 65 m × 80 （满分 80；D_eff = 躯干质心沿赛道方向累计有效前程，不计回退，上限 65 m）
时间加成 = clamp(20 × (1 − (T − 360) / 360), 0, 20) （满分 20；T ≤ 360s 满分，≥ 720s 为 0；未完赛 T = 720s）
罚则检测方式分值
越出赛道边界质心 |y| > 2.5m，每次越界−3 / 次
足部刮蹭（scuff）踝部水平力 > 2× 垂直力，超出前 5 次起算−1 / 次（无封顶）
跌倒（非终止）躯干质心 < 0.3m 持续 >1s 或膝/肘/头触地−3 / 次

机型：人形 Oli EDU（得分权重 ×1.0）。单局限时 720s，跌倒不终止、可自主恢复。

04 / 提交方式

命令行提交，自动评测上榜

提交唯一通道是命令行 submit.py：交推理代码 + 策略权重（solution.py + policy.pt），带队伍令牌。分数由主办方评测算出，榜单取该队历史最高分。两道赛题共用同一个 submit.py，按 --competition 区分。

① 准备工具

下载提交工具 submit.py。<队伍令牌> 由主办方私发。令牌可能以 - 开头，命令里务必用等号形式 --token=<队伍令牌>（否则会被解析成选项）。

# 下载提交工具（单文件，零依赖） wget https://limx-troncamp.github.io/troncamp-web-locomotion/submit.py

② 提交

按赛题选 --competition 与 --robot：Tron 用 sfyg_tron2a（腿式）或 wfyg_tron2a（轮式），人形用 oli。

# 提交 TronCamp（TRON 2 腿式；轮式把 robot 换成 wfyg_tron2a） python3 submit.py --server http://118.196.31.68:18080 --token=<队伍令牌> \ --competition tron --robot sfyg_tron2a \ --ckpt-file ./policy.pt --code-dir solution_dir # solution_dir（含 solution.py 推理代码的文件夹）
# 提交 Humanoid（人形 Oli） python3 submit.py --server http://118.196.31.68:18080 --token=<队伍令牌> \ --competition humanoid --robot oli \ --ckpt-file ./policy.pt --code-dir solution_dir # solution_dir（含 solution.py 推理代码的文件夹）

③ 查询

查本队各次提交的状态与分数（queued / done / failed + 总分）。

# 查询本队提交状态与分数 python3 submit.py --server http://118.196.31.68:18080 --token=<队伍令牌> --status

05 / 评测

计分：线上 70% + 完赛 30%

最终成绩由两部分构成：线上排行榜实测分与完赛材料评审分。线上分由评测集群自动产出，完赛材料由主办方评审。

总评 = 线上排行榜 × 70% + 完赛材料 × 30% （线上：评测集群实测总分；完赛：完整代码思路讲解 + 实机 / 仿真演示视频）

完赛材料

请于 2026 年 7 月 15 日 24:00 前，将最终讲解及演示视频提交至邮箱 troncamp@limxdynamics.com，该视频将作为完赛材料计分依据。

  • 完整展示代码思路及实机 / 仿真演示过程
  • 建议配合画外音讲解，便于主办方理解方案亮点
  • 整体时长请控制在 3 分钟以内
  • 邮件命名格式：姓名-专属提交码-赛道名称（示例：张三-Token-Locomotion）
